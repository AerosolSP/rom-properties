This copy of minizip-1.1 is a modified version of the original.

The following changes have been made to the original:

- The build system has been replaced with CMake.

- The additional MiniZip tools and examples have been removed.

- The Windows Runtime "app" check has been fixed.

- iowin32.c: Use the specified dwMoveMethod instead of hard-coding
  FILE_CURRENT.

To obtain the original minizip-1.1, visit
http://www.winimage.com/zLibDll/minizip.html

NOTE: Some changes were imported from zlib-1.2.9's copy of
minizip-1.1. To obtain a copy of zlib-1.2.9, visit the zlib
website at http://www.zlib.net/ .
